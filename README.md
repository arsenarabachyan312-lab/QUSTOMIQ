# Handoff: QUSTOMIQ — редизайн сайта (концепт «Q-Core»)

## Overview
Премиальный корпоративный сайт IT-компании **QUSTOMIQ**: кастомная разработка, системные интеграции и AI-решения для бизнеса (рынок — Россия и СНГ). Задача — превратить текущую «сайт-визитку» в полноценный многосекционный премиальный сайт на основе утверждённого визуального концепта **«Q-Core»**.

## About the Design Files
Файлы в этом пакете — **дизайн-референсы, сделанные на HTML/CSS/Canvas**. Это эталон внешнего вида и поведения, **а не продакшн-код для копирования**. Задача — **пересоздать этот дизайн в вашем стеке** (рекомендуется Next.js + TypeScript + Tailwind; если стек уже есть — используйте его паттерны). HTML показывает точные цвета, типографику, отступы и анимации — ориентируйтесь на них, но пишите идиоматичный код своего фреймворка.

## Fidelity
**High-fidelity.** Цвета, шрифты, размеры и анимации — финальные. Hero нужно воспроизвести максимально близко к референсу. Остальные секции описаны как структура + токены (см. ниже) — собираются в той же визуальной системе.

## Ключевая идея бренда
Название = **Custom + IQ + Q**. Фирменный знак — кольцо буквы **Q** с «хвостиком», внутри которого живёт **ядро интеллекта**: анимированная нейро-сеть из частиц (canvas). Вокруг Q по орбите медленно вращаются чипы систем клиента: **CRM, 1С, ERP, API**. Метафора: системы клиента вращаются вокруг интеллекта, собранного под него. Это центральный, узнаваемый и «непохищаемый» элемент бренда — он должен жить в герое и в логотипе.

### Typography
- **Display / заголовки:** `Space Grotesk` (700). H1 — 72px / line-height .98 / letter-spacing −2.5px.
- **Body / UI:** `Sora` (400–600). Подзаголовок 18.5px / 1.6.
- **Mono / метки, теги, чипы, RU·EN:** `JetBrains Mono` (400–600), часто uppercase + letter-spacing.
- Все три — Google Fonts.

### Spacing & radius
- Горизонтальные поля секции: `56px` (десктоп).
- Радиусы: кнопки-таблетки `50px`; крупные кнопки/карточки `12–22px`; чипы `50px`.
- Тени карточек: `0 6px 20px rgba(12,40,55,.12)` (мягкая), `0 24px 60px rgba(14,40,52,.13)` (крупная).

### Анимации
- `qfloat` — кольцо Q плавно качается по Y (±14px, 7s, ease-in-out).
- `qspin` — орбита вращается 360°, 38s linear infinite.
- `qspinrev` — чипы крутятся в обратную сторону (−360°, 38s), чтобы текст оставался читаемым.
- Нейро-ядро (canvas): 34 частицы, броуновское движение, линии между близкими (<56px), радиальный cyan-градиент по центру. Цикл через `requestAnimationFrame`.
- `prefers-reduced-motion: reduce` — все анимации выключаются, ядро рендерится статичным кадром.

## Interactions & Behavior
- **Тема light/dark** — переключатель, сохранение в `localStorage`, без вспышки (read до hydration / inline-скрипт).
- **RU / EN** — i18n; весь копирайт из словаря.
- **AI по отделам** — клик по сегменту меняет активный ключ, карточка ре-маунтится с короткой анимацией появления (fade + translateY ~12px, .4s).
- **Hover:** кнопки слегка темнеют/светлеют; карточки — небольшой подъём тени.
- **Reduced motion** — останавливает qspin/qfloat и анимацию ядра.

## State Management
- `theme: 'light' | 'dark'` (persist).
- `locale: 'ru' | 'en'`.
- `activeDept` для секции AI (default «Продажи»).
- Остальное — статический контент из словаря.
